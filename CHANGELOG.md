# Changelog

All notable changes to the Personal OS project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2025-03-09

### Added
- Professional README with author bio and achievements
- CONTRIBUTING.md with comprehensive contribution guidelines
- CODE_OF_CONDUCT.md following Contributor Covenant v2.1
- CHANGELOG.md to track project evolution
- GitHub issue templates for better community engagement
- Enhanced badges showcasing tech stack and certifications
- Professional developer advocate branding

### Changed
- Updated README to highlight Akshata Sawant's developer advocacy work
- Improved documentation structure with better navigation
- Enhanced project metadata and social links

## [1.1.0] - 2025-01-15

### Added
- Search functionality across all tasks
- Keyboard shortcuts for common actions (Ctrl/Cmd+K for quick capture)
- Enhanced task management with bulk operations
- Task filtering and sorting improvements

### Changed
- Improved UI responsiveness on mobile devices
- Optimized performance for large task lists
- Better error handling for calendar sync

### Fixed
- Calendar sync timeout issues
- Drag and drop glitches on Firefox
- Task persistence edge cases

## [1.0.0] - 2024-12-20

### Added
- Rule of 3 daily focus feature
- Master Kanban board with 4 columns
- Drag and drop task management
- Synergy tags for multi-role creators
- Quick capture brain dump area
- Hub filtering (Tech/AI, Career, Travel)
- Google Sign-In authentication
- Google Calendar integration
- Local storage persistence
- Dark mode UI with glassmorphism effects
- Responsive design (mobile, tablet, desktop)

### Technical Additions
- React 18.3 with hooks
- Vite 5.1 build tool
- Tailwind CSS 3.4 styling
- Firebase Authentication
- Google Calendar API integration
- @hello-pangea/dnd for drag and drop
- lucide-react for icons

### Documentation
- Complete setup guide (SETUP_GUIDE.md)
- API keys guide (GET_API_KEYS.md)
- Vercel deployment guide (DEPLOY_TO_VERCEL.md)
- Project structure documentation (PROJECT_STRUCTURE.md)
- Comprehensive README with usage instructions

## [0.2.0] - 2024-11-15

### Added
- Firebase hosting configuration
- Netlify deployment support
- Environment variables management
- OAuth consent screen setup guide

### Changed
- Improved onboarding flow
- Enhanced Google Calendar sync reliability
- Better error messages for authentication issues

### Fixed
- Firebase authentication edge cases
- Calendar event creation bugs
- Local storage quota exceeded handling

## [0.1.0] - 2024-10-01

### Added
- Initial project setup
- Basic React application structure
- Tailwind CSS configuration
- Firebase project setup
- Basic authentication flow
- Simple task management (no drag and drop yet)
- Initial README and documentation

### Technical Setup
- Vite development environment
- ESLint configuration
- Git repository initialization
- MIT License
